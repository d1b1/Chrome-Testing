Test are made up of steps. Each step has three options, name, jquery selector(s), action and value.

Name + JQuery Selector(s) + Action + Value = DOM Assertion.

The 'name' is used to define the expected action. Use anything you want to make the test suite steps readable.

The 'selectors' are simple selectors available in JQuery. If you can find an element in using jQuery in the console you can use it in a step.

  • $('#id') Find with a given Id.
  • $('#id.classname') Find with a given Id and class.
  • $('.classname') Find with a given classname.
  • $('tr.classname') Find with a Table Tr with a given classname.

The 'action' is what you want to perform. These are simple statement, that either change the URL, or trigger an event on a given element. For example, find a button with a given class, and click it.

How to build a step? Open our browser and either navigate to a page or url, and right click on a given element in the DOM. You will see 'step building' option that let you grab the classes, id and value for use in the curent test suite steps.
